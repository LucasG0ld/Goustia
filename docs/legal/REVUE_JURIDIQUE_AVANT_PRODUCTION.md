# Revue juridique et conformité avant production

Statut : **actions humaines non exécutées**.

- [ ] Identifier l’éditeur, le directeur de publication et l’hébergeur.
- [ ] Faire valider politique de confidentialité, CGU et mentions légales.
- [ ] Confirmer les bases légales, notamment pour les données alimentaires.
- [ ] Déterminer si une AIPD est obligatoire et la réaliser si nécessaire.
- [ ] Confirmer toutes les durées de conservation et mettre en place les purges.
- [ ] Signer et archiver les DPA de chaque sous-traitant.
- [ ] Vérifier les transferts hors EEE et leurs garanties.
- [ ] Définir la procédure de réponse aux droits et le canal de contact.
- [ ] Vérifier la configuration des traceurs avec l’outil CNIL approprié.
- [ ] Vérifier la qualification et les obligations éventuelles de données de
      santé.
- [ ] Décider si un DPO est requis ou utile.
- [ ] Faire valider l’avertissement nutritionnel et de sécurité alimentaire.
- [ ] Remplacer tous les champs `[à compléter]` dans les pages publiques.
- [ ] Publier des versions définitives et recueillir à nouveau les acceptations.
